Comment doit être la vidéo avant l'import dans flash :
- mov ou avi avec codec pas trop récent et surtout pas sorenson, 
pas trop destructif : Microsoft video 1 par exemple
- 25 images par secondes
- taille : peut importe
- durée : coupe vidéo faite avant l'import

______________________

On doit dont transformer les vidéos au format swf.

Lors de l'import dans flash :
> Qualité haut débit : 
taux de qualité sorenson : 85, 1:1

> Qualité bas débit :
taux de qualité sorenson : 85, 1:2 (1:3 marche bien aussi pour des petites videos)

Procédure de numérisation de dvd
1- copier les VOB sur le disque dur
2- Avec le logiciel NeoDivX : créer un d2v (son non séparé) avec tous les vob nécessaires
3- Avec le logiciel NeoDivX encoder la vidéo : 
Codec video : MpegXDiv4, 
Son : pas d'amplifcation ni de normalisation, compression 96kbps
Lancer l'encodage
4- avec le logiciel virtual dub :
Recadrer la vidéo
Fixer des dimensions plus petites
Retailler la longueur de la video
Choisir le mode de compression Microsoft Video 1
Exporter la video
5- importer l'avi créée dans flash en fixant les dimensions finales.
